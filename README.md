## NPM

Run: 
```bash
sudo npm install pm2 -g
npm i pm2 tmijs ws fs spotify-buddylist string-similarity nodemon dotenv node-fetch discord.js
```

## GIT 

Run: 
```bash
git clone https://github.com/Mr-Cheeezz/MainsBot.git
```

## START

Run:
```bash
cd MainsBot && pm2 start app.js && cd ..
```
